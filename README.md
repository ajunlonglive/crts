# crts

A real-time-simulation game written in c.

[website](https://mochiro.moe/crts)

# build

```
$ git submodule init
$ git submodule update
$ meson build
$ ninja -C build
```

# usage

First start the server, then start the client[s].

## server

```
$ bin/s -h
```

## client

```
$ bin/c -h
```
