A real time simulation game and engine written from scratch in c11
Features:
  • custom opengl renderer
  • real time shadows and reflections
  • custom linear algebra library (no glm)
  • advanced terrain generator
  • efficent pathfinding
  • custom sound engine
  • file format loaders for assets (wav, obj, tga)
  • immediate mode ui framework
  • fully customizeable vim-like keymap system
  • text-mode (curses) ui
  • real time networking with a custom reliable udp protocol
  • serialized data is compressed with a custom arithmetic encoder
